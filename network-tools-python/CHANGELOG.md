# 更新日志

本文档记录“网络运维工具箱软件”的主要版本变化。

## V1.0 - 2026-06-06

### 新增

- Tkinter 桌面端图形界面。
- Ping 连通性检测。
- DNS 域名解析。
- TCP 端口与端口范围扫描。
- 局域网设备扫描。
- 公网 IP 与 IP 归属地查询。
- 网络延迟和下载速度测试。
- 从 `targets.txt` 读取目标进行批量检测。
- TXT、CSV、HTML、Markdown 报告导出。
- 软件版本信息和 MIT 许可证。

### 优化

- 兼容中文和英文 Windows Ping 输出。
- 增强空输入、错误端口、无效地址、网络超时和权限不足处理。
- 第三方测速库不可用时自动使用标准库降级方案。
- GUI 网络任务使用后台线程执行，避免界面无响应。
