# 用户使用手册

## 软件信息

- 软件名称：网络运维工具箱软件
- 英文名称：Network Tools Python
- 当前版本：V1.0
- 作者：Simpson Liao
- 适用系统：Windows 10/11

## 启动图形界面

```bash
python main.py gui
```

在目标输入框中填写 IP 地址或域名，在端口输入框中填写单个端口、逗号分隔端口或端口范围。

示例：

```text
baidu.com
80,443,3306
8000-8010
```

## 功能说明

### Ping 检测

检测目标是否可达，并显示发送数、接收数、丢包率和平均延迟。

### 端口扫描

通过 TCP 连接检测端口是否开放。扫描结果受目标防火墙和网络策略影响。

### DNS 解析

解析域名对应的 IPv4 或 IPv6 地址，并显示解析耗时。

### IP 查询

输入公网 IP 后查询国家、地区、城市和运营商。输入框留空时查询本机公网 IP。

### 网络测速

优先使用 `speedtest-cli`。依赖未安装或服务不可用时，自动使用 Ping 和 HTTPS 下载方式降级测试。

### 报告导出

GUI 中完成检测后点击“报告导出”，可选择 TXT、CSV、HTML 或 Markdown。

## 使用限制

- 只应检测自己拥有或获得授权的设备和网络。
- 网络受限时，IP 查询和下载测速可能失败。
- 端口显示关闭不代表设备一定离线。
